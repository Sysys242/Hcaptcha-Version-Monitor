### ⚙️ Installation

- Requires: `Python 3.8+`, `Requests Pip Module Installed`
- Start: `py main.py`

- Send To Discord Webhook
- Automatic Monitoring
- Simple & Easy To Setup
- Print to console
- Proxyless
